# 项目概述
名称：基于GoJS实现相似度树
描述：背景是后端进行相似度匹配，需要一个可视化树去展示匹配结果。
1. 节点代表一张图片和对应指向最相似的那个节点
2. 每个节点能够直接预览显示迷你的缩略图
3. 树的节点方向是从子节点指向父节点
4. 指向的这个线条中间，需要显示子节点的相似度，（相对于父）
5. 可以选中一个节点，去切换父节点（人为修改）
6. 相似度越高，节点连线越粗，最大粗度4px，最小粗度1px
7. 如果节点被人为修改，线条颜色需要变成黄色
8. 当相似度大于0.8时，线条颜色需要变成绿色
9. 线条默认颜色为灰色
10. 可以滚动、拖拽整个画布，查看完整树结构
你需要使用vue3 + gojs + pnpm + element-plus来实现这个项目
1. 页面结构
- 左侧画布（占比范围大，80%以上）
- 预览区，点击节点，预览该节点的图片，显示id\相似度\父节点id，可以编辑修改父节点。

后端数据结构
```typescript
interface Node {
    id: string
```
