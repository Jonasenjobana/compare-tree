# GoJS 替换为 AntV G6 实现方案
## 现有功能梳理（需100%复现）
1. 垂直方向树布局，参数同现有：
   - 节点排布方向向下（angle:90）
   - 层间距80px，节点间距40px
   - 正交连线带5px圆角
2. 节点样式：
   - 80x100px圆角矩形，白色背景灰色边框
   - 上方72x40.5px图片区域，下方显示节点selfId文本
   - 选中时显示3px蓝色边框
3. 连线样式：
   - 线宽根据score动态计算
   - 颜色根据score和isModified动态计算
   - 箭头指向父节点，不被节点遮挡
   - 连线上显示score保留两位小数的文本
4. 交互功能：
   - 节点click事件触发nodeSelect emit
   - 节点doubleClick事件触发nodeDoubleClick emit
   - 鼠标悬浮节点时，所有关联连线置顶显示
   - 支持缩放、拖拽平移画布
   - nodes属性变化自动更新视图
   - 对外暴露getDiagram方法返回实例
## 实现步骤
### 步骤1：安装依赖
安装@antv/g6最新版本
### 步骤2：新增g6-tree组件（完全不修改原有gojs-tree组件）
1. 新增`src/components/g6-tree/index.vue`组件，对外props、emit、defineExpose接口和gojs-tree完全一致，父组件可以零成本替换使用
2. 实现G6树图初始化：
   - 配置画布大小自适应容器
   - 配置树布局参数和现有完全一致
   - 开启缩放、平移交互
3. 自定义节点模板，完全复现现有样式和选中效果
4. 自定义连线模板，复现动态线宽、动态颜色、箭头样式、分数文本显示
5. 实现事件绑定：click、dblclick、mouseenter、mouseleave事件逻辑和现有完全一致
6. 实现watch监听nodes变化，自动更新数据
### 步骤3：保留原有功能完全不受影响
原有gojs-tree组件完全不动，gojs依赖也保留，父组件可以自由切换使用两个组件，不会出现功能失效问题
### 步骤4：功能验证
运行项目验证所有功能和原有效果完全一致
