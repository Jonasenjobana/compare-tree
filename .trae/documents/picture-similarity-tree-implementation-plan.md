# 图片相似度树实现计划
## 项目概述
基于Vue3 + GoJS + Element Plus实现图片相似度可视化树，展示后端相似度匹配结果，支持交互编辑父节点关联。
## 实现步骤
### 1. 项目初始化与基础配置
- 检查当前项目结构，若未初始化则使用PNPM创建Vue3 + Vite项目
- 安装核心依赖：gojs、element-plus、@element-plus/icons-vue
- 配置Element Plus按需引入/全局引入，配置项目基础路径别名
### 2. 项目结构与类型定义
- 规范目录结构：
  - src/components: 公共组件（GoJS画布组件、预览面板组件）
  - src/types: TypeScript类型定义
  - src/utils: 公共工具方法（相似度转线宽、颜色判断逻辑等）
  - src/styles: 公共样式变量
- 定义节点数据类型，完全对齐后端返回结构：
  ```typescript
  interface Node {
    id: string
    parentId: string
    selfId: string
    score: number
    matchDate: string
    imageUrl: string
    isModified?: boolean // 前端扩展字段，标记是否手动修改过父节点
  }
  ```
### 3. GoJS画布核心组件实现（左侧80%+区域）
- 初始化GoJS Diagram实例，配置画布基础属性：
  - 支持拖拽平移、滚轮缩放、框选等交互
  - 配置树布局方向，实现子节点指向父节点的连接逻辑
- 自定义节点模板：
  - 每个节点展示图片缩略图 + 节点id文本
  - 支持选中状态高亮
- 自定义连线模板：
  - 连线中间显示相似度数值文本
  - 动态线宽逻辑：相似度越高线越粗，范围1px-4px
  - 动态颜色逻辑：默认灰色，相似度>0.8时绿色，手动修改关联后黄色
### 4. 右侧预览面板实现
- 选中节点后展示：
  - 大图预览
  - 节点信息：selfId（图片自身id）、当前相似度score、当前父节点parentId、匹配时间matchDate
  - 父节点编辑下拉选择框，支持切换父节点
- 提交修改后自动更新画布中对应连线的状态（标记为手动修改，颜色变为黄色）
### 5. 交互逻辑整合
- 节点选中事件同步到预览面板，更新展示内容
- 父节点修改事件同步到画布，更新树结构和连线样式
- 画布交互事件监听，确保操作流畅无卡顿
### 6. 样式与适配优化
- 整体页面布局：左侧占比85%，右侧占比15%，适配不同屏幕尺寸
- 使用Element Plus组件样式规范，统一交互体验
- 公共样式抽离，主题色使用变量定义，避免硬编码
### 7. 功能验证测试
- 验证所有需求点：
  - 节点缩略图展示正常
  - 连线相似度显示、线宽、颜色规则正确
  - 父节点修改功能正常，修改后连线变黄
  - 画布拖拽、缩放、平移正常
  - 预览面板信息展示和编辑功能正常
## 验收标准
所有require.md中列出的14项需求全部实现，交互流畅无bug。
